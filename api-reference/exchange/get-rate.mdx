---
title: Get Rate
api: GET https://pay-api.holdstation.com/api/v1/exchange/rate
description: Retrieve the current buy and sell exchange rates between USD and VND.
---

## Query Parameters

| Parameter | Type | Description |
|---|---|---|
| `original` | boolean | Set to `true` to get the rate before fees are applied |

## Response Fields

| Field | Type | Description |
|---|---|---|
| `rate.buy` | string | Exchange rate for buying tokens with VND (onramp) |
| `rate.sell` | string | Exchange rate for selling tokens to VND (offramp) |

```json 200
{
  "message": "success",
  "data": {
    "rate": {
      "buy": "25750",
      "sell": "25000"
    }
  }
}
```
