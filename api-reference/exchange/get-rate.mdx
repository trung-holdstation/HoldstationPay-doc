---

## Overview

Partners use this API to retrieve the buy and sell exchange rates between USD/VND in the system.

<ParamField query="original" type="boolean">
  Set to `true` to get the original rate before applying fees.
</ParamField>

<ResponseExample>
```json 200
{
  "message": "success",
  "data": {
    "rate": {
      "buy": "25850",
      "sell": "25750"
    }
  }
}
```
</ResponseExample>
