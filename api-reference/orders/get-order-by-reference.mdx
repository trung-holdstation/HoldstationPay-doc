---
title: Get Order by Reference
api: GET https://pay-api.holdstation.com/partners/orders/reference-no/{reference_no}
description: Query a payment order by its partner reference number.
---

## Path Parameters

| Parameter | Type | Required | Description |
|---|---|---|---|
| `reference_no` | string | ✅ | The partner's reference number assigned when creating the order |

## Response Fields

Same order fields as [Create Deposit](/api-reference/orders/create-deposit) / [Create Withdrawal](/api-reference/orders/create-withdrawal), wrapped in a list:

| Field | Type | Description |
|---|---|---|
| `orders` | array | List of matching order objects (typically one) |
| `total_count` | integer | Total number of matching orders |
