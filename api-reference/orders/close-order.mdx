---
title: "Close Order"
api: "POST https://pay-api.holdstation.com/partners/orders/{order_id}/close"
description: "Close a payment order that is in Processing state with a Pending processing state."
---

## Overview

Partners can close a payment order that is in the `Processing` state and `Pending` processing state by providing the `order_id`. Once closed, the order cannot be paid.

<ParamField path="order_id" type="string" required>
  The unique UUID of the order to close.
</ParamField>

<ResponseExample>
```json 200
{
  "message": "success",
  "data": {
    "success": true,
    "code": 0,
    "error": ""
  }
}
```

```json 400
{
  "error": "invalid request",
  "error_code": "BAD_REQUEST",
  "message": "Order cannot be closed in its current state"
}
```

```json 401
{
  "error": "unauthorized",
  "error_code": "UNAUTHORIZED",
  "message": "Invalid or missing API key"
}
```
</ResponseExample>
