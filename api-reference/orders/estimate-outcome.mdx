---
title: Estimate Outcome
api: POST https://pay-api.holdstation.com/api/v1/payments/estimate-outcome
description: Calculate the final amount a user will receive for a given input. Set reverse=true to calculate the required input for a desired outcome.
---

## Request Body

| Field | Type | Required | Description |
|---|---|---|---|
| `amount` | string | ✅ | Input amount. In VND for deposit, USD for withdrawal |
| `order_type` | integer | ✅ | `1` = Deposit (onramp), `2` = Withdrawal (offramp) |
| `chain_id` | integer | ✅ | Blockchain network ID — see [List Tokens](/api-reference/tokens/list-tokens) |
| `token_address` | string | ✅ | Token contract address — see [List Tokens](/api-reference/tokens/list-tokens) |
| `reverse` | boolean | ✅ | If `true`, calculates required input for a desired outcome |
| `buffer` | number | ✅ | Rate buffer override (fractional). Effective buffer = max(override, configured). Example: `0.005` = 0.5% |

## Response Fields

| Field | Type | Description |
|---|---|---|
| `outcome` | string | Estimated amount the user will receive after fees |
| `rate` | number | Exchange rate used for this estimate |
| `fee_vnd` | number | Total estimated fee in VND |
| `fees.service_fee` | number | Service fee in VND |
| `fees.min_fee` | number | Minimum fee applied in VND |

```json 200
{
  "message": "success",
  "data": {
    "outcome": "100",
    "rate": 25750,
    "fee_vnd": 5000,
    "fees": {
      "service_fee": 5000,
      "min_fee": 2000
    }
  }
}
```
