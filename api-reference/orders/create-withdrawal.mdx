## Overview

Partner uses this API to create a new withdrawal order. See [Offramp Flow](/guides/offramp-flow) for the full integration guide.

<ParamField body="amount" type="string" required>
  Amount to withdraw in USD (or VND if `currency=vnd`).
</ParamField>

<ParamField body="currency" type="string" required>
  Currency of the amount: `vnd` or `usd`.
</ParamField>

<ParamField body="chain_id" type="integer" required>
  Blockchain network ID. See [List Tokens](/api-reference/tokens/list-tokens).
</ParamField>

<ParamField body="token_address" type="string" required>
  Token contract address. See [List Tokens](/api-reference/tokens/list-tokens).
</ParamField>

<ParamField body="payment_info" type="object" required>
  User's bank account details to receive VND.

  <Expandable title="payment_info">
    <ParamField body="bank_id" type="integer" required>
      Bank ID. See [List Banks](/api-reference/banks/list-banks).
    </ParamField>
    <ParamField body="full_name" type="string" required>
      Full name of the account holder.
    </ParamField>
    <ParamField body="account_type" type="integer" required>
      `1` = Bank account number, `2` = ATM card number.
    </ParamField>
    <ParamField body="account_number" type="string" required>
      Bank account number.
    </ParamField>
  </Expandable>
</ParamField>

<ParamField body="callback" type="string" required>
  HTTPS URL to receive order status webhook notifications.
</ParamField>

<ParamField body="reference_no" type="string" required>
  Partner's own reference number for tracking.
</ParamField>

<ParamField body="buffer" type="number" required>
  Optional buffer rate override (fractional). Effective buffer = max(override, configured). Example: `0.005` = 0.5%.
</ParamField>
