## Overview

Create a new withdrawal order to sell tokens for VND. The user will transfer tokens to the provided recipient address, and Holdstation Pay will send VND to the user's bank account.

<ParamField body="amount" type="string" required>
  Amount to withdraw in USD.
</ParamField>

<ParamField body="currency" type="string" required>
  Currency of the amount: `vnd` or `usd`.
</ParamField>

<ParamField body="chain_id" type="integer" required>
  Blockchain network ID.
</ParamField>

<ParamField body="token_address" type="string" required>
  Token contract address.
</ParamField>

<ParamField body="callback" type="string" required>
  Callback URL for receiving withdrawal updates via [webhooks](/guides/webhooks).
</ParamField>

<ParamField body="reference_no" type="string" required>
  Reference number for the withdrawal.
</ParamField>

<ParamField body="buffer" type="number" required>
  Optional buffer rate override (fractional). If set, effective buffer = max(override, configured buffer). Example: `0.005` = 0.5%.
</ParamField>

<ParamField body="payment_info" type="object" required>
  Payment information for the withdrawal.

  <Expandable title="payment_info properties">
    <ParamField body="bank_id" type="integer" required>
      Bank ID from the [List Banks](/api-reference/banks/list-banks) endpoint.
    </ParamField>
    <ParamField body="full_name" type="string" required>
      Full name of the account holder.
    </ParamField>
    <ParamField body="account_type" type="integer" required>
      Type of account number. `1` = Bank account number, `2` = ATM card number.
    </ParamField>
    <ParamField body="account_number" type="string" required>
      Bank account number.
    </ParamField>
  </Expandable>
</ParamField>
