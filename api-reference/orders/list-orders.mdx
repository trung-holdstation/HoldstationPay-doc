## Overview

Partners can query a list of their payment orders with optional filtering by state, type, and date range.

<ParamField query="type" type="integer[]">
  Filter by order type. `1` = Deposit, `2` = Withdrawal.
</ParamField>

<ParamField query="state" type="integer[]">
  Filter by order state. `1` = New, `2` = Processing, `3` = Completed, `4` = Failed, `5` = Closed.
</ParamField>

<ParamField query="processing_state" type="integer[]">
  Filter by processing state. Deposit: `10`–`16`. Withdrawal: `20`–`25`. See table below.
</ParamField>

<ParamField query="page" type="integer">
  Page number for pagination.
</ParamField>

<ParamField query="page_size" type="integer">
  Number of results per page.
</ParamField>

<ParamField query="sort_by" type="string">
  Sort field. Available values: `created_at`, `expired_at`, `amount`.
</ParamField>

<ParamField query="sort_desc" type="boolean">
  If `true`, sort in descending order.
</ParamField>

<ParamField query="created_at_from" type="integer">
  Filter orders created after this Unix timestamp (seconds).
</ParamField>

<ParamField query="created_at_to" type="integer">
  Filter orders created before this Unix timestamp (seconds).
</ParamField>

<ParamField query="expired_at_from" type="integer">
  Filter orders expiring after this Unix timestamp (seconds).
</ParamField>

<ParamField query="expired_at_to" type="integer">
  Filter orders expiring before this Unix timestamp (seconds).
</ParamField>

<ParamField query="recipient" type="string">
  Filter by token recipient wallet address.
</ParamField>

<ParamField query="reference_nos" type="string[]">
  Filter by a list of partner reference numbers.
</ParamField>

## Processing State Values

| Value | Name | Description |
|---|---|---|
| `10` | DepositFiatPending | Waiting for user to deposit fiat |
| `11` | DepositFiatConfirmed | Fiat deposit confirmed |
| `12` | DepositFiatFailed | Fiat deposit failed |
| `13` | DepositCryptoPending | Waiting for USDT transfer on-chain |
| `14` | DepositCryptoConfirmed | USDT transfer confirmed |
| `15` | DepositCryptoFailed | USDT transfer failed |
| `16` | DepositFiatWaitingForApproval | Fiat payment pending manual approval |
| `20` | WithdrawCryptoPending | Waiting for user to transfer USDT |
| `21` | WithdrawCryptoConfirmed | USDT transfer confirmed on-chain |
| `22` | WithdrawCryptoFailed | USDT transfer failed |
| `23` | WithdrawFiatPending | Waiting for VND bank transfer |
| `24` | WithdrawFiatConfirmed | VND bank transfer confirmed |
| `25` | WithdrawFiatFailed | VND bank transfer failed |
