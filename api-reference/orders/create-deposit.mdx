title: "Create Deposit"
api: "POST https://pay-api.holdstation.com/partners/orders/deposit"
description: "Create a new onramp order for a user to buy tokens by depositing VND via bank transfer."
---

## Overview

Partner uses this API to create a new on-chain token transfer order. See [Onramp Flow](/guides/onramp-flow) for the full integration guide.

<ParamField body="amount" type="string" required>
  Amount to deposit in VND (or USD if `currency=usd`).
</ParamField>

<ParamField body="currency" type="string" required>
  Currency of the amount: `vnd` or `usd`.
</ParamField>

<ParamField body="chain_id" type="integer" required>
  Blockchain network ID. See [List Tokens](/api-reference/tokens/list-tokens).
</ParamField>

<ParamField body="token_address" type="string" required>
  Token contract address. See [List Tokens](/api-reference/tokens/list-tokens).
</ParamField>

<ParamField body="recipient" type="string" required>
  User's wallet address to receive the tokens.
</ParamField>

<ParamField body="callback" type="string" required>
  HTTPS URL to receive order status webhook notifications.
</ParamField>

<ParamField body="reference_no" type="string" required>
  Partner's own reference number for tracking.
</ParamField>

<ParamField body="buffer" type="number" required>
  Optional buffer rate override (fractional). Effective buffer = max(override, configured). Example: `0.005` = 0.5%.
</ParamField>
