---
title: "Set Partner Webhook URL"
api: "POST https://pay-api.holdstation.com/partners/webhook-url"
description: "Set (or clear) the partner-level webhook URL that receives Holdstation Pay events."
---

Set or clear the partner-level webhook URL. Holdstation Pay sends a `POST` request to this URL for every event addressed to the partner — including order status updates and fund balance changes.

| Event topic                         | When it fires                                                                                          | Reference |
|-------------------------------------|--------------------------------------------------------------------------------------------------------|-----------|
| `pay.order.status-updated`          | An order's state or processing state changes.                                                          | [Orders Webhook](/guides/webhooks/orders-webhook) |
| `pay.partner.fund-balance-changed`  | The partner's prefund balance changes (deposit, withdraw, or compensation).                            | [Balance Change Webhook](/guides/webhooks/balance-change-webhook) |

Pass an empty string `""` as `webhook_url` to disable notifications.

<Note>
  Each order may also override the destination by passing its own `callback` field at creation time. When an order-level callback is set, order events for that order go there instead of the partner-level URL.
</Note>

## Authentication

<ParamField header="Partner-App-Key" type="string" required>
  Your partner app key.
</ParamField>

<ParamField header="X-HSPay-Signature" type="string" required>
  Base64-encoded Ed25519 signature of the sign data.
</ParamField>

<ParamField header="X-HSPay-Timestamp" type="string" required>
  Unix timestamp (seconds) of the request. Must be within 60 seconds of server time.
</ParamField>

## Body Parameters

<ParamField body="webhook_url" type="string" required>
  HTTPS URL to receive partner events. Pass an empty string `""` to clear the webhook.
</ParamField>

<RequestExample>
```json Set
{
  "webhook_url": "https://yourapp.com/api/webhooks/hspay"
}
```

```json Clear
{
  "webhook_url": ""
}
```
</RequestExample>

<ResponseExample>
```json 200
{
  "message": "success",
  "data": {
    "webhook_url": "https://yourapp.com/api/webhooks/hspay"
  }
}
```

```json 400
{
  "error": "invalid request",
  "error_code": "BAD_REQUEST",
  "message": "webhook_url must be a valid HTTPS URL"
}
```
</ResponseExample>
