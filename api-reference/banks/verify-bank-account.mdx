---
title: Verify Bank Account
api: POST https://pay-api.holdstation.com/api/v1/payment-methods/verify
description: Verify a bank account number and retrieve account holder details.
---

## Request Body

| Field | Type | Required | Description |
|---|---|---|---|
| `bin_code` | string | ✅ | Bank Identification Number (BIN) code — see [List Banks](/api-reference/banks/list-banks) |
| `account_number` | string | ✅ | Bank account number to verify |

## Response Fields

| Field | Type | Description |
|---|---|---|
| `bank` | object | Bank information |
| `bank.id` | integer | Unique bank ID |
| `bank.bank_name` | string | Full bank name |
| `bank.short_name` | string | Bank short name / abbreviation |
| `bank.bin_code` | string | Bank Identification Number (BIN) code |
| `bank.image` | string | URL to the bank logo image |
| `bank.created_at` | object | Record creation time — `{ seconds, nanos }` |
| `bank.updated_at` | object | Record last updated time — `{ seconds, nanos }` |
| `account_name` | string | Account holder's full name |
| `account_number` | string | Verified bank account number |
