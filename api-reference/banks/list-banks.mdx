---
title: List Banks
api: GET https://pay-api.holdstation.com/api/v1/banks
description: Retrieve the list of banks supported for fiat withdrawals.
---

## Query Parameters

| Parameter | Type | Description |
|---|---|---|
| `page` | integer | Page number for pagination |
| `size` | integer | Number of results per page |
| `ids` | integer[] | Filter by specific bank IDs |

## Response Fields

| Field | Type | Description |
|---|---|---|
| `banks` | array | List of bank objects |
| `banks[].id` | integer | Unique bank ID |
| `banks[].bank_name` | string | Full bank name |
| `banks[].short_name` | string | Bank short name / abbreviation |
| `banks[].bin_code` | string | Bank Identification Number (BIN) code |
| `banks[].image` | string | URL to the bank logo image |
| `banks[].created_at` | object | Record creation time — `{ seconds, nanos }` |
| `banks[].updated_at` | object | Record last updated time — `{ seconds, nanos }` |
| `page` | integer | Current page number |
| `page_size` | integer | Number of items per page |
| `total_count` | integer | Total number of banks available |
