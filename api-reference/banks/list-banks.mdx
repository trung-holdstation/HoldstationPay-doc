---
title: List Banks
openapi: GET /api/v1/banks
---

## Query Parameters

| Parameter | Type | Required | Description |
|---|---|---|---|
| `page` | integer | No | Page number for pagination |
| `size` | integer | No | Number of results per page |
| `ids` | integer[] | No | Filter by specific bank IDs |

## Response

```json
{
  "message": "success",
  "data": {
    "banks": [
      {
        "id": 42,
        "bank_name": "Tien Phong Commercial Joint Stock Bank",
        "short_name": "TPBank",
        "bin_code": "970423",
        "image": "https://static.holdstation.com/banks/tpbank.png",
        "created_at": { "seconds": 1700000000, "nanos": 0 },
        "updated_at": { "seconds": 1700000000, "nanos": 0 }
      }
    ],
    "page": 1,
    "page_size": 20,
    "total_count": 50
  }
}
```

| Field | Type | Description |
|---|---|---|
| `banks` | array | List of bank objects |
| `page` | integer | Current page number |
| `page_size` | integer | Number of items per page |
| `total_count` | integer | Total number of banks available |

### Bank Object

| Field | Type | Description |
|---|---|---|
| `id` | integer | Unique bank ID |
| `bank_name` | string | Full bank name |
| `short_name` | string | Bank short name / abbreviation |
| `bin_code` | string | Bank Identification Number (BIN) code |
| `image` | string | URL to the bank logo image |
| `created_at` | object | Record creation timestamp — `{ seconds, nanos }` |
| `updated_at` | object | Record last updated timestamp — `{ seconds, nanos }` |
